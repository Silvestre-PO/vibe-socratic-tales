## ğŸ“– Overview

**Socratic Tales** is a multimodal educational engine where a child's drawing becomes the teacher.

Unlike standard AI chatbots, we orchestrate a **Multi-Model Architecture**. We use the reasoning depth of **Gemini 2.5 Pro** to construct pedagogical metaphors (Socratic Method) and the speed of **Gemini 2.5 Flash** to generate real-time audio and visuals.

It transforms a static photo of a doodle into an interactive, voice-acted story that teaches science through experience, not facts.

We utilize specific models for specific cognitive and sensory tasks. This specialized routing ensures the highest quality reasoning with the lowest latency for media generation.

## ğŸš€ How It Works

1.  **Input:** User takes a photo of a drawing + asks a voice question.
2.  **Orchestration (The Brain):** `gemini-2.5-pro` analyzes the image style and the question. It generates a structured **JSON** storyboard.
3.  **Parallel Generation:**
    -   `gemini-2.5-flash-image` receives the visual prompt to render the next scene.
    -   `gemini-2.5-flash-preview-tts` receives the script to generate the audio narration.
4.  **Output:** The React frontend assembles these assets into an instant story.
